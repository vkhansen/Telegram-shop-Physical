## Goal

Share referral; apply codes. **Physical invite cards** (CARD-41): numbered tear-off QR cards + brand A4 PDF + post-distribution name-by-index registry.

---

## Preconditions

- Referral / reference codes enabled (default on)
- For invite PDF: brand row exists; `BOT_USERNAME` set for deep links

---

## Entry points

| Surface | Handler / CLI |
|---------|----------------|
| Customer referral UI | `referral_system` · `reference_code_handler` |
| `/start CODE` deep link | `user/main.py` → `try_register_with_reference_code` |
| Admin PDF / registry / assign | `admin/reference_code_management` |
| CLI | `bot_cli.py invite-cards …` |

---

## Conversation (Telegram as-built)

| Step | Behaviour |
|------|-----------|
| New user `/start` | Language → if payload is invite code, auto-redeem; else prompt type code |
| Existing user | Main menu |
| Admin sheet | Pick brand → count → codes in DB → PDF (+ .tex) |
| Admin assign | `7 Ali` lines → `recipient_name` + `given_at` on `card_number` |

---

## Physical card data model

| Field | Meaning |
|-------|---------|
| `card_number` | Printed index on **both** halves (unique) |
| `code` | 8-letter invite / start payload |
| `card_batch_id` | Print run |
| `brand_id` | Brand theme / filter |
| `recipient_name` | Name written on stub after hand-out |
| `given_at` | When name was logged |

Spec: [CARD-41](../../done/CARD-41-physical-invite-cards-brand-pdf.md)

---

## State machine

```text
referral / register
  → language
  → (deep-link code | typed code)
  → use_reference_code
  → privacy / main menu

admin_invite
  → sheet (brand + count) | registry | assign names
```

---

## Branches / errors

| Invalid / expired / own / max-uses code | error + re-prompt |
| Deep-link payload not accepted | message + fall back to typed entry |
| Assign unknown `#` | error line in bulk result |

---

## Platform masks

| Platform | Support | Notes |
|----------|---------|-------|
| telegram | full | deep link + admin PDF + registry |
| instagram | off | masked shop package |
| line | off | Flex / text; no ops |
| web | varies | public/commerce API; no invite-card print |
